Description of the analytical solutions.

The files X_calling.m contains example of how to use the analytical solution.

1) MilanTWM1D.m
    Solution of the hyperbolic heat equation in time-domain with temperature as boundary condition in T(y = 0) = Tc and T(y = H) = Ts.
    
2) D1_BHE_f.m
    Solution of the pennes bio-heat equation in time-domain with temperature as boundary condition in T(y = 0) = Tc and T(y = H) = Ts.
    
3) D1_HEAT_f.m
    Solution of the heat equation in time-domain with temperature as boundary condition in T(y = 0) = Tc and T(y = H) = Ts.
